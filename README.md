# 알고리즘  테스트

## ❓**제작목적**

본레포지토리는 코딩테스트를 준비하면서 배웠던것을 기록하고 각 문제별로 깨달았것을 기록하는 용도

그리고 다시 추후에 문제를 맞이했을때 과거와 현재의 생각차이를 스스로 깨닫고싶은 용도
